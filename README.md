# lezec.cz scraper

1. Přejmenuj `.env.example` na `.env`

```bash
mv .env.example .env
```

2. Uprav přihlašovací údaje v `.env`

```
LEZEC_USER=tvuj_login
LEZEC_PASS=tvoje_tajne_heslo
```

3. Nainstalujte závislosti

```bash
yarn
```

4. Spusťte

```bash
yarn dev
```

5. V souboru `climbs.json` jsou stažené všechny tvé přelezy
